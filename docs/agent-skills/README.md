Guides for agents helping Srinidhi **write, revise, and publish** on this Hugo site.

## Skill files

| File | Use when... |
|------|-------------|
| `authoring-workflow.md` | Running Hugo commands to create/preview content |

| `content-review.md` | Reviewing a draft for clarity, structure, tone |
| `ai-writing-detection.md` | Detecting AI-like patterns and humanizing drafts |
| `publishing-checklist.md` | Final checks before merging to `main` |
| `style-guide.md` | Markdown/Hugo formatting questions |
